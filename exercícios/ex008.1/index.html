Pricipais Formatações

Negrito / Destaque

Pode-se usar a tag semântica < strong > ou a tag não-semântica < b >. Ambas oferecem o mesmo efeito de negrito, porém o funcionamento é diferente.

Veja um exemplo de uso práratico da tag <strong> :
O negrito está aqui

Veja um exemplo de uso práratico da tag <b> :
Temos o negrito nesse trecho.

Itálico / Ênfase

Pode-se usar a tag semântica < em > ou a tag não-semântica < i >. Ambas oferecem o mesmo efeito de itálico, porém o funcionamento é diferente.

Veja um exemplo de uso prático da tag <em> :
O itálico está aqui

Veja um exemplo de uso prático da tag <i> :
Temos o itálico nesse trecho.

Texto Marcado

Pode-se usar a tag <mark> para deixar um: texto marcado.

Texto pequeno

Pode-se usar a tag <small> para deixar um: texto pequeno.

Texto cortado

Pode-se usar a tag <del> para deixar um: texto cortado/deletado.

Texto sobrescrito

Pode-se usar a tag <sup> para "elevar" um expressão como em: ax2+bx+c.

Texto subescrito

Pode-se usar a tag <sub> para deixar uma expressão em forma de índice, como em: H2O.
