Outras Formatações

Código-Fonte

Serve para manter uma estética de programação, usando a tag <code>

Pré-Fomatação

Serve para conservar a estrutura de um código na exibição do site, usando a tag <pre>. Como por exemplo em: 


    algoritmo "semnome"
    var:
        num: vetor[1,2,3] de inteiro
    inicio:

Citações Simples

Usar aspas numa frase recuada em relação a margem esquerda, usando a tag <q>. Exemplo: como meu avô dizia: água mole em pedra dura, tanto bate até que fura.

Citação Completa

Fazer uma citação longa, usando a tag <blockquote>. Por exemplo:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Eligendi blanditiis sit veritatis error debitis! Inventore, dignissimos natus? Odio, veniam quos aliquam voluptas error et corrupti placeat inventore quas tenetur nisi!

Abreviações

Oferecer explicação de siglas ao pousar o cursor sobre a sigla, usando a tag <abbr>. Como por exemplo:

PS2
